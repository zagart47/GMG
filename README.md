## Getting Started

### Dependencies

* Go 1.16+
* Windows, Linux or Mac OS
* MinGW-W64 for building

### Installing

```
git clone https://github.com/zagart47/GMG.git
cd GMG
go build .
```

### Executing program

```
go run .
```
